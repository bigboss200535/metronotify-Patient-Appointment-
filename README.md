## Metronotify (Patient Appointment System)
Metronotify is a lightweight web application designed to help patient Book appointments with Doctors. 
It provides an organized workflow for recording transactions, tracking appintments statuses, and storing patient information in a centralized and easy-to-use interface.

## Features
- Patient Management
Add, edit, and organize patient profiles.

- Appointment  Tracking
Record Appointment, update their status, and maintain a clear historical log.

- Search & Filtering
Quickly locate patient or appointment records through built-in search tools.

- Role-Based Access
Manage permissions for developers, administrators, doctors, or patient and data entry clerks.

## Usage
- Navigate to the dashboard to view patient and appointment summaries.
